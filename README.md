## ğŸ“Œ **Project Overview**

This project analyzes **HR data from a Multinational Corporation (MNC)** containing **2 million employee records**.
The goal is to extract actionable insights on:

* Workforce distribution
* Salary patterns
* Attrition trends
* Employee performance
* Work mode preferences
* Department-wise analysis

Using **Python (Pandas, NumPy, Matplotlib, Seaborn)**, we performed detailed EDA and answered critical HR business questions that support **strategic HR decision-making**.
